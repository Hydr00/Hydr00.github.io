XOR is a logical analysis type of two operators, symbolized as XOR or EOR. Unlike general logic or OR, it is true when the two values are the same and the values are different. XOR cryptography (simple XOR cipher) is a simple encryption algorithm in cryptography, which refers to the XOR operation of information to achieve the purpose of encryption and decryption. According to this logic, each character of the text serial can be encrypted by bitwise XOR with a given key. If you want to decrypt, you only need to perform a bitwise XOR operation between the encrypted result and the key again.

Introduction

XOR operation (XOR) is an important bit operation based on binary in computer programming language, which has important practical value for improving program readability and running efficiency. XOR operation has the common properties of exchange law, combination law, reflexivity law, pairing and so on, which is widely used in parity discrimination, error correction, network coding, information encryption, image processing and so on.

In cryptography, frequency analysis refers to the frequency at which letters or combinations of letters appear in the text. Classical codes can be cracked by using frequency analysis.

Frequency analysis is based on the principle that different letters or combinations of letters appear at different frequencies in any written language. Moreover, for any piece of text written in this language, there is roughly the same distribution of characteristic letters. For example, in English, the letter E appears very frequently, while the letter X appears less frequently. Similarly, the frequency of two-letter combinations such as ST, NG, TH, and QU is very high, while the combination of NZ and QJ is very few.
